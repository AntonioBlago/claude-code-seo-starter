# Client knowledge base

The **read-first** source material for this client. Before any new deliverable,
Claude reads this folder so analyses build on what exists instead of duplicating it.

| Folder | What goes here |
|---|---|
| `audits/` | Prior SEO/SEA audits, tech reports (PDF, docs) |
| `keywords/` | Target keyword exports (`.xlsx`, `.csv`) — SV, positions, themes |
| `calls/` | Call notes & transcripts — goals, constraints, pain points |
| `offers/` | Previous offers and contracts — scope already sold |
| `brand/` | Client logo, CI, tone-of-voice (if provided) |

**Rule:** drop raw material in here as you receive it. Deliverables go in dated
`YYYY-MM-DD_<Task>/` folders one level up, never in `_knowledge/`. Nothing here is
committed to git.
