# SEO Offer / Proposal

Turn analysis + client context into a **phased, priced offer**.

## Inputs to gather first

- Status-Quo and Potential outputs for the domain.
- Existing audits, call notes, prior offers.
- Client context: budget, internal resources, decision makers, current tools, pain points.

## Structure

- **Phase 0** — Setup & data review (explicitly recognise existing work; never duplicate it).
- **Phase 1** — Audit (SEO + optional SEA assessment).
- **Phase 2** — Strategic accompaniment + enablement (monthly).
- **Phase 3** — Optional advanced module (e.g. Neuro-SEO / GEO).

For each phase: scope, deliverables, timeline, investment.

## Management-ready add-ons

- Executive summary (English if the HQ is international).
- Quarterly milestone plan, KPQ/KPI framework.
- ROI section tied back to the Potential numbers.

## Output

`clients/<domain>/YYYY-MM-DD_Offer/offer_<date>.md` → then `/pdf-build` for the PDF.
Methodology: `docs/workflows.md`. Slash command: `/angebot <domain>`.
